# HR Attrition SQL Analysis

## About
SQL analysis on IBM HR Analytics dataset (1,470 employees) 
to identify key drivers of employee attrition.

## Tools Used
- MySQL Workbench
- SQL (Joins, Aggregations, CTEs, Window Functions)
- Currently upskilling in Python (Pandas , NumPy)
  
## Dataset
IBM HR Analytics Employee Attrition Dataset

## Key Findings
- Employees who left earned 30% less than those who stayed
- Overtime employees had 3x higher attrition rate
- Sales department had highest attrition rate (20.6%)
- Laboratory Technician was highest at-risk job role (62 employees)

## Queries Covered
1. Basic data exploration
2. Total employees by department
3. Attrition count by department
4. Attrition rate % by department
5. Top 3 job roles by attrition
6. Average salary - Left vs Stayed
7. Overtime impact on attrition
8. CTE - High risk employee identification
